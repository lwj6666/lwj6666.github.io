字体图标的浅浅了解

发表于 2022-07-20 更新于 2022-08-03 分类于 学习路程

关于字体图标

iconfont(字体图标)使用来实现图标管理和统一。其有以下几种优势:

1. 矢量图标: 放大缩小不会糊照片。
2. 轻量级: 体积小、渲染块、降低服务器请求次数(所有图标只用一个请求)
3. 灵活性:
可以灵活的修改样式,例如尺寸、颜色等(如同修改字体一般)
4. 兼容性: 几乎兼容所有主流浏览器

使用字体图标的方法

使用步骤：

  1. 进入官方网址并注册账号(github账号通用)。
  2. 进入素材库中的官方图标库(也可以用其他创作者的图标,前提是适配)。
  3. 将选中的图标加入购物车中,最后一同添加进项目(自己根据所写网页创建项目)中一同下载。
  4. 在将项目中的图标下载至本地。
  5. 最后根据demo网页里的文档进行使用。

iconfont官方网址：阿里巴巴矢量图标库

(根据demo_index文档使用即可)

字体格式文件的比较

  • “iconfont.svg”是图标库的svg格式文件。(可用svg文件上传到阿里巴巴矢量图标库转换成字体图标)
  • “iconfont.eot”，“iconfont.ttf”,“iconfont.woff”是图标库的字体格式文件。

各字体格式文件介绍

SVG
SVG（Scalable Vector Graphics），即矢量图，是用于描述二维矢量图形的一种图形格式。在 Web 中使用 SVG 可以解决位图放大失真的问题。

TTF / OTF
TTF（TrueTypeFont）是Apple公司和Microsoft公司共同推出的字体文件格式，随着windows的流行，已经变成最常用的一种字体文件表示方式。部分的因为这种格式容易被非法复制，因此催生了后来的WOFF字体格式。

EOT
EOT（Embedded Open Type），是微软创造的字体格式。这种格式只在IE6-IE8里使用。

WOFF
WOFF（Web Open Font Format）是一种网页所采用的字体格式标准。此字体格式发展于2009年，现在正由万维网联盟的Web字体工作小组标准化，以求成为推荐标准。此字体格式不但能够有效利用压缩来减少档案大小，并且不包含加密也不受DRM（数位著作权管理）限制。WOFF字体通常比其它字体加载要快，因为使用了OTF和TTF字体里的存储结构和压缩算法。这种字体格式还可以加入元信息和授权信息。

WOFF2(*)
WOFF2（Web Open Font Format 2.0），相比woff最大的优化应该是加强了字体的压缩比。

字体图标与精灵图对比

  1. 字体图标在动画、多色展示上不如精灵图
  2. 字体图标在高分辨率、变换颜色等效果、文件体积上比精灵图更优秀
