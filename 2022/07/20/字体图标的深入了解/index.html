е…ідәҺеӯ—дҪ“еӣҫж Ү

iconfont(еӯ—дҪ“еӣҫж Ү)дҪҝз”ЁжқҘе®һзҺ°еӣҫж Үз®ЎзҗҶе’Ңз»ҹдёҖгҖӮе…¶жңүд»ҘдёӢеҮ з§ҚдјҳеҠҝ:

1. зҹўйҮҸеӣҫж Ү: ж”ҫеӨ§зј©е°ҸдёҚдјҡзіҠз…§зүҮгҖӮ
2. иҪ»йҮҸзә§: дҪ“з§Ҝе°ҸгҖҒжёІжҹ“еқ—гҖҒйҷҚдҪҺжңҚеҠЎеҷЁиҜ·жұӮж¬Ўж•°(жүҖжңүеӣҫж ҮеҸӘз”ЁдёҖдёӘиҜ·жұӮ)
3. зҒөжҙ»жҖ§:
еҸҜд»ҘзҒөжҙ»зҡ„дҝ®ж”№ж ·ејҸ,дҫӢеҰӮе°әеҜёгҖҒйўңиүІзӯү(еҰӮеҗҢдҝ®ж”№еӯ—дҪ“дёҖиҲ¬)
4. е…је®№жҖ§: еҮ д№Һе…је®№жүҖжңүдё»жөҒжөҸи§ҲеҷЁ

дҪҝз”Ёеӯ—дҪ“еӣҫж Үзҡ„ж–№жі•

дҪҝз”ЁжӯҘйӘӨпјҡ

  1. иҝӣе…Ҙе®ҳж–№зҪ‘еқҖе№¶жіЁеҶҢиҙҰеҸ·(githubиҙҰеҸ·йҖҡз”Ё)гҖӮ
  2. иҝӣе…Ҙзҙ жқҗеә“дёӯзҡ„е®ҳж–№еӣҫж Үеә“(д№ҹеҸҜд»Ҙз”Ёе…¶д»–еҲӣдҪңиҖ…зҡ„еӣҫж Ү,еүҚжҸҗжҳҜйҖӮй…Қ)гҖӮ
  3. е°ҶйҖүдёӯзҡ„еӣҫж ҮеҠ е…Ҙиҙӯзү©иҪҰдёӯ,жңҖеҗҺдёҖеҗҢж·»еҠ иҝӣйЎ№зӣ®(иҮӘе·ұж №жҚ®жүҖеҶҷзҪ‘йЎөеҲӣе»әйЎ№зӣ®)дёӯдёҖеҗҢдёӢиҪҪгҖӮ
  4. еңЁе°ҶйЎ№зӣ®дёӯзҡ„еӣҫж ҮдёӢиҪҪиҮіжң¬ең°гҖӮ
  5. жңҖеҗҺж №жҚ®demoзҪ‘йЎөйҮҢзҡ„ж–ҮжЎЈиҝӣиЎҢдҪҝз”ЁгҖӮ

iconfontе®ҳж–№зҪ‘еқҖпјҡйҳҝйҮҢе·ҙе·ҙзҹўйҮҸеӣҫж Үеә“

(ж №жҚ®demo_indexж–ҮжЎЈдҪҝз”ЁеҚіеҸҜ)

еӯ—дҪ“ж јејҸж–Үд»¶зҡ„жҜ”иҫғ

  • вҖңiconfont.svgвҖқжҳҜеӣҫж Үеә“зҡ„svgж јејҸж–Үд»¶гҖӮ(еҸҜз”Ёsvgж–Үд»¶дёҠдј еҲ°йҳҝйҮҢе·ҙе·ҙзҹўйҮҸеӣҫж Үеә“иҪ¬жҚўжҲҗеӯ—дҪ“еӣҫж Ү)
  • вҖңiconfont.eotвҖқпјҢвҖңiconfont.ttfвҖқ,вҖңiconfont.woffвҖқжҳҜеӣҫж Үеә“зҡ„еӯ—дҪ“ж јејҸж–Үд»¶гҖӮ

еҗ„еӯ—дҪ“ж јејҸж–Үд»¶д»Ӣз»Қ

SVG
SVGпјҲScalable Vector GraphicsпјүпјҢеҚізҹўйҮҸеӣҫпјҢжҳҜз”ЁдәҺжҸҸиҝ°дәҢз»ҙзҹўйҮҸеӣҫеҪўзҡ„дёҖз§ҚеӣҫеҪўж јејҸгҖӮеңЁ Web дёӯдҪҝз”Ё SVG еҸҜд»Ҙи§ЈеҶідҪҚеӣҫж”ҫеӨ§еӨұзңҹзҡ„й—®йўҳгҖӮ

TTF / OTF
TTFпјҲTrueTypeFontпјүжҳҜAppleе…¬еҸёе’ҢMicrosoftе…¬еҸёе…ұеҗҢжҺЁеҮәзҡ„еӯ—дҪ“ж–Үд»¶ж јејҸпјҢйҡҸзқҖwindowsзҡ„жөҒиЎҢпјҢе·Із»ҸеҸҳжҲҗжңҖеёёз”Ёзҡ„дёҖз§Қеӯ—дҪ“ж–Үд»¶иЎЁзӨәж–№ејҸгҖӮйғЁеҲҶзҡ„еӣ дёәиҝҷз§Қж јејҸе®№жҳ“иў«йқһжі•еӨҚеҲ¶пјҢеӣ жӯӨеӮ¬з”ҹдәҶеҗҺжқҘзҡ„WOFFеӯ—дҪ“ж јејҸгҖӮ

EOT
EOTпјҲEmbedded Open TypeпјүпјҢжҳҜеҫ®иҪҜеҲӣйҖ зҡ„еӯ—дҪ“ж јејҸгҖӮиҝҷз§Қж јејҸеҸӘеңЁIE6-IE8йҮҢдҪҝз”ЁгҖӮ

WOFF
WOFFпјҲWeb Open Font FormatпјүжҳҜдёҖз§ҚзҪ‘йЎөжүҖйҮҮз”Ёзҡ„еӯ—дҪ“ж јејҸж ҮеҮҶгҖӮжӯӨеӯ—дҪ“ж јејҸеҸ‘еұ•дәҺ2009е№ҙпјҢзҺ°еңЁжӯЈз”ұдёҮз»ҙзҪ‘иҒ”зӣҹзҡ„Webеӯ—дҪ“е·ҘдҪңе°Ҹз»„ж ҮеҮҶеҢ–пјҢд»ҘжұӮжҲҗдёәжҺЁиҚҗж ҮеҮҶгҖӮжӯӨеӯ—дҪ“ж јејҸдёҚдҪҶиғҪеӨҹжңүж•ҲеҲ©з”ЁеҺӢзј©жқҘеҮҸе°‘жЎЈжЎҲеӨ§е°ҸпјҢе№¶дё”дёҚеҢ…еҗ«еҠ еҜҶд№ҹдёҚеҸ—DRMпјҲж•°дҪҚи‘—дҪңжқғз®ЎзҗҶпјүйҷҗеҲ¶гҖӮWOFFеӯ—дҪ“йҖҡеёёжҜ”е…¶е®ғеӯ—дҪ“еҠ иҪҪиҰҒеҝ«пјҢеӣ дёәдҪҝз”ЁдәҶOTFе’ҢTTFеӯ—дҪ“йҮҢзҡ„еӯҳеӮЁз»“жһ„е’ҢеҺӢзј©з®—жі•гҖӮиҝҷз§Қеӯ—дҪ“ж јејҸиҝҳеҸҜд»ҘеҠ е…Ҙе…ғдҝЎжҒҜе’ҢжҺҲжқғдҝЎжҒҜгҖӮ

WOFF2(*)
WOFF2пјҲWeb Open Font Format 2.0пјүпјҢзӣёжҜ”woffжңҖеӨ§зҡ„дјҳеҢ–еә”иҜҘжҳҜеҠ ејәдәҶеӯ—дҪ“зҡ„еҺӢзј©жҜ”гҖӮ
