网站的诞生

发表于 2022-02-23 更新于 2022-07-11 分类于 blog_deployer

简介

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。所以可以通过github + Hexo框架 来搭建个人博客

搭建博客所需要的工具

  • git的安装
  • Node.js的安装

安装后进入控制台安装hexo：

Ps:windowsx系统注意要进入管理者权限下进行运行（cmd）

搭建io仓库

在github上创建一个***.github.io为名的仓库

生成属于自己的SSH Keys

  • 生成ssh

  • ssh建立
    找到用户中的id_rsa.pub文件，将里面的东西放到github setting里新建的ssh里面生成Key

  • 进行ssh测试

本地生成博客内容

  • 创建一个文件夹

  • 在博客文件夹内进入git控制台并输入一下命令

将博客搭建到github上

修改博客文件夹中的_config.yml文件

deploy:    
    type: git  
    repository: 你的仓库地址
    branch: main

安装hexo自动部署

进入博客文件中的git控制台： 

npm install hexo-deployer-git --save       
hexo g  生成静态文件
hexo d  部署

初次部署以此需要：

  1. 输入用户名
  2. 输入密码
  3. 输入令牌

markdown初体验

发表于 2022-02-09 更新于 2022-05-12

好用的编写工具：Vcode + markdown all in one 插件

特点：

  1. 提供了常用操作便利的快捷键
  2. 支持目录
  3. 一边书写一边预览
    • Ctrl + Shift + v （同页面打开）
    • Ctrl + K然后在按V （侧页面打开）
  4. 可轻松转换为HTML文件和PDF文件
  5. 优化了List editing的编辑
  6. 可格式化table (Alt + Shift + F) 以及Task list (use Alt + C to check/uncheck a list item)
  7. 支持特殊数学符号渲染

Ctrl + Shift + P    指令台
Ctrl + B    粗体
Ctrl + I    斜体
Alt + S    删除线
Ctrl + Shift + ]    标题(uplevel)
Ctrl + Shift + [    标题(downlevel)
Ctrl + M    切换数学环境
Alt + C    Check/Uncheck task list item（选中或则取消任务列表项）

Markdown的写作手法(对我自己能用上的汇总)

标题

使用#号可表示1-6级标题，一级标题对应一个#号，二级标题对应两个#号，以此类推。

#
##
###
####
#####
######

段落

  • 粗体：在文字两边分别加**
  • 斜体：在文字两边分别加*
  • 斜粗体：在文字两边分别加*
  • 分割线：使用三个或三个以上的-或者*都可以
  • 删除线：在文字两边分别加两个波浪号(~)
  • 下划线：在文字两边添加HTML的<u>标签来实现

列表

无序列表使用*、+或是-作为列表标记，这些标记后面要添加一个空格，然后再填写内容：

* 打游戏
- 打游戏
+ 打游戏

有序列表使用数字并加上.号来表示：

1. 打游戏
2. 打羽毛

列表嵌套

1. 做运动
    - 打羽毛球
    - 跑步

引用

要创建块引用，请在段落前添加一个 > 符号。 

> #### The quarterly results look great!
>
> - Revenue was off the chart.
> - Profits were higher than ever.
>
>  *Everything* is going according to **plan**.

代码

1.如果是段落上的一个函数或片段的代码可以用反引号(`)把它包起来:

`printf`函数

2.代码区块：段落前后加三个反引号(`) 或者缩进至少四个空格或一个制表符。
